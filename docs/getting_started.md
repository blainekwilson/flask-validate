# Getting Started

Install (from source while package is unpublished):

```bash
pip install -e .
```

Run the example application:

```bash
python examples/example_on_error.py
```

Open `http://127.0.0.1:5000/demo_on_error` to try the demo UI.

Notes:

- This library is lightweight by design — suitable for small Flask apps and utilities.
- For production API services consider schema-first validation frameworks.
